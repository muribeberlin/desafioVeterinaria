# desafioVeterinaria

Desafío correspondiente a la semana 1

La veterinaria Santa Margarita necesita construir un sistema para llevar las fichas médicas de sus pacientes. Los datos que necesita registrar son:
-Nombre paciente
-Tutor
-Fecha nacimiento
-Especie
-Raza
-Género
-Está esterilizado
-Peso
-Diagnóstico
